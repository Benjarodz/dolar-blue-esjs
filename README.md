# 💵 Dólar Blue ESJS

Aplicación web desarrollada en JavaScript que permite consultar la cotización del Dólar Blue en Argentina mediante autenticación con DNI y palabra secreta.

## 🚀 Características

- 🔐 Autenticación mediante DNI y palabra secreta.
- 💲 Consulta de la cotización actual del Dólar Blue.
- ⚡ Interfaz simple y responsiva.
- 🖥️ Desarrollado con HTML, CSS y JavaScript (ES Modules).

## 📂 Estructura del proyecto

```
├── css/
├── js/
├── img/
├── index.html
└── README.md
```

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES Modules)
- Fetch API

## ▶️ Cómo ejecutar el proyecto

1. Clona el repositorio:

```bash
git clone https://github.com/Benjarodz/dolar-blue-esjs.git
```

2. Entra al directorio del proyecto:

```bash
cd dolar-blue-esjs
```

3. Abre `index.html` con un servidor local.

Por ejemplo, con VS Code puedes utilizar la extensión **Live Server**.
